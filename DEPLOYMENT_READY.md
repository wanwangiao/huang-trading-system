## 🚀 立即部署指南

### 1. 安裝新的依賴
```bash
cd chengyivegetable/veg_delivery_db_updated
npm install
```

### 2. 配置環境變數
複製 `.env.example` 為 `.env` 並填入真實值：
```bash
cp .env.example .env
# 編輯 .env 文件，填入您的配置
```

### 3. 本地測試
```bash
npm start
```
訪問 http://localhost:3000 查看效果

### 4. 部署到 Render
- 推送代碼到 GitHub
- 在 Render 建立新的 Web Service
- 設定環境變數
- 點擊 Deploy
